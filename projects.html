Historical Weather Visualization

Single Page Web App • Javascript D3 Visualizations View the story full-screen here.

This project came about as my younger sister was applying to colleges and my older sister's partner was applying to residency. Due to Covid, neither could visit all of the locations they wanted to. So, I created a historical weather visualization so they could get a (virtual) taste of what the weather is like in each of the locations.

This project uses a Weatherwheel class created by Eric Lo and data from the World Weather API. It taught me best practices when modifying classes created by others and the benefits of creating single page web applications for interactive visualizations.
